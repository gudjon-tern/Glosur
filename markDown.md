##Escape
 ```sh
\   backslash
`   backtick
*   asterisk
_   underscore
{}  curly braces
[]  square brackets
()  parentheses
#   hash mark
+   plus sign
-   minus sign (hyphen)
.   dot
!   exclamation mark
```

For example:
This
 ```sh
 \\ \` \* \_ \{ \} \[ \] \( \) \# \+ \- \. \!
```
returns this \\ \` \* \_ \{ \} \[ \] \( \) \# \+ \- \. \!
